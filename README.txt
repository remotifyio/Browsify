Browsify is a MIDI-mappable browser for Ableton Live. v0.1.0

Map all of the controls you need for complete midi control
Navigate, Preview, Search, and load into your Ableton project — directly from your MIDI controller.

Select the search box from your midi controller for quick filtering

Navigation can also be done with a knob, encoder or fader style control. Browsify supports Absolute, relative and pitch bend inputs for this.
